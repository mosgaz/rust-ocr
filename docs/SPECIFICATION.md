# Техническое задание: rust-ocr - Автоматизированный OCR для размножения учебных пособий

## 1. Общие сведения

**Название проекта**: `rust-ocr`  
**Версия**: `v0.2.0`  
**Дата**: 22 февраля 2026  
**Статус**: **Готов к продакшену** (100% функциональность реализована)

## 2. Цель проекта

**Автоматизировать процесс** преобразования отсканированных страниц учебных пособий из формата **JPG/PNG** в **чистый редактируемый текст** для массового размножения копий.

**Бизнес-задача**: Сократить время перепечатывания учебников с **тысяч часов вручную** до **5 минут автоматизации**.

## 3. Требования к функциональности

### 3.1. Входные данные
| Параметр | Требование |
|----------|------------|
| **Формат** | JPG, JPEG, PNG |
| **Размер** | 300-5000px (автоматическое масштабирование) |
| **Язык** | Русский (rus) + английский (eng) |
| **Качество** | Сканированные учебники (шум, неровности допустимы) |

### 3.2. Выходные данные
| Параметр | Требование |
|----------|------------|
| **Формат** | Чистый текст (.txt) |
| **Структура** | Блоки текста + пустые строки между абзацами |
| **Точность** | **≥90%** для печатного текста |
| **Блоки** | **≥19 блоков на страницу** (page1.jpg = 21 блок) |

### 3.3. Технические характеристики
| Компонент | Технология | Параметры |
|-----------|------------|-----------|
| **OCR** | Tesseract | PSM=6, DPI=300, lang="rus" |
| **Предобработка** | image crate | Resize 1800px, +20 яркость, +25 контраст |
| **CLI** | clap | `ocr-scan input.jpg -o output.txt` |
| **Конфигурация** | .env | OCR + Enhance параметры |

## 4. Функциональные требования

### 4.1. CLI интерфейс
```
./target/release/ocr-scan input.jpg -o output.txt [--aggressive]
```

### 4.2. Batch обработка (скрипты)
| Скрипт | Функция | Пример |
|--------|---------|--------|
| `batch_ocr.sh` | Массовая обработка `scans/*` → `results/*.txt` | `./scripts/batch_ocr.sh` |
| `single_ocr.sh` | Один файл | `./scripts/single_ocr.sh page1.jpg` |
| `enhance_batch.sh` | Улучшение изображений | `./scripts/enhance_batch.sh` |

### 4.3. Конфигурация (.env)
```env
OCR_TESSERACT_LANG=rus
OCR_TESSERACT_PSM=6
OCR_TESSERACT_DPI=300
ENHANCE_RESIZE_WIDTH=1400
ENHANCE_BRIGHTNESS=20
ENHANCE_CONTRAST=25
ENHANCE_SHARPEN_RADIUS=0.8
```

## 9. Эксплуатация

### 9.1. Развертывание
```bash

chmod +x scripts/*.sh
```

### 9.2. Пайплайн размножения
```bash
cp учебник/*.jpg scans/
./scripts/batch_ocr.sh
cat results/*.txt > учебник.txt
pandoc учебник.txt -o учебник.docx  # Опционально
```
